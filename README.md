# Rust Programming Journey 🦀
Chào mừng bạn đến với repository ghi lại hành trình học tập Rust của mình! Tại đây, mình sẽ lưu trữ các bài thực hành, code ví dụ và các dự án nhỏ từ khóa học **Rust Programming Basics** trên *[Cyfrin Updraft]*.

## 📚 Mục tiêu khóa học
* **Cơ bản:** Nắm vững cú pháp, biến, hàm và luồng điều khiển trong Rust.
* **Chuyên sâu:** Hiểu sâu về *Ownership*, *Borrowing* và *Lifetimes*.
* **Ứng dụng:** Xây dựng smart contracts trên **Solana** và **Arbitrum Stylus**.

---

## 🛠 Cách sử dụng (Cargo Commands)

Để làm việc với các project trong repo này, bạn hãy sử dụng các lệnh của [Cargo](https://doc.rust-lang.org/cargo/) tại Terminal của VS Code:

| Lệnh          | Mô tả                                                      |
| :------------ | :--------------------------------------------------------- |
| `cargo run`   | Biên dịch và chạy project hiện tại                         |
| `cargo fmt`   | Tự động định dạng code theo chuẩn Rust                     |
| `cargo test`  | Chạy các unit test trong project                           |
| `cargo check` | Kiểm tra lỗi cú pháp mà không cần biên dịch ra file binary |

## 🚀 Cách chạy dự án
Để chạy các đoạn code trong repo này, đảm bảo bạn đã cài đặt Rust. Sau đó, trong thư mục dự án, bạn có thể sử dụng các lệnh Cargo quen thuộc:

```
Bash

```
Kiểm tra lỗi định dạng code:

```
Bash

```

```
Bash
cargo test
```
